About GeneWeb format

GeneWeb (.gw) is a text-based genealogy format with the following structure:

  • Families start with "fam" followed by parents and children
  • Persons are defined with their names, dates, and events
  • Dates can be in various formats (DD/MM/YYYY, approximate dates, etc.)
  • The file will be converted to GEDCOM format before import
