# Legacy — Modernization of GeneWeb (OCaml) in Python

This repository contains the **Python reimplementation** of the **GeneWeb** application (originally in OCaml), as part of the *Awkward Legacy* project.

## 🎯 Goal
- Reproduce the business logic
- Apply a professional test policy, quality standards, and documentation.
- Provide a modern deployment strategy (Docker, CI/CD).
